# ChatClone – WhatsApp Chat Mimic

A web application that analyzes exported WhatsApp chats and generates an AI capable of mimicking a person’s communication style. The app features a **realistic WhatsApp-like UI** and enables **real-time AI conversations**.

---

## Features

- **WhatsApp Chat Analysis**: Upload `.txt` chat exports; automatically cleans timestamps, system messages, and media notices.  
- **AI Persona Generation**: Creates an AI that mimics the chat partner’s tone, style, and emoji usage.  
- **Real-Time Conversations**: Chat interactively with the AI persona.  
- **Authentic UI**: WhatsApp-like interface with chat bubbles, timestamps, typing indicators, and profile headers.  
- **Lightweight & Fast**: Pure frontend implementation using HTML, CSS, and JavaScript.
